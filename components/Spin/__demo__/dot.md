## zh-CN

通过 `dot`属性，可以展示点类型的指示符。

## en-US

The dot type indicator can be displayed by specifying the `dot` property.

```js
import { Spin } from '@arco-design/web-react';

const App = () => {
  return <Spin dot />;
};

export default App;
```
