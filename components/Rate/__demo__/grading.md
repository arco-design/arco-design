## zh-CN

通过 `grading` 指定使用笑脸分级。

## en-US

Use `grading` to show score with smiley icons.

```js
import { Rate } from '@arco-design/web-react';

const App = () => {
  return (
    <div>
      <Rate grading />
    </div>
  );
};

export default App;
```
