## zh-CN

输入页码，可快速跳转到指定页。

## en-US

Enter the page number to quickly jump to the specified page.

```js
import { Pagination } from '@arco-design/web-react';

const App = () => {
  return <Pagination total={200} showJumper />;
};

export default App;
```
